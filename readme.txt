[OFFLINE]
Tools:: Laravel 5.8, xampp (local server), phpmyadmin, browser
1. Extract file recipes.zip 
2. Aktifkan Apache dan MySQL pada Xampp
3. Buatlah database di phpmyadmin bernama dbrecipes
4. Lalu import file sql (dbrecipes.sql) yang berada pada hasil extract file yang tadi (recipes-sql)
5. Jalankan pada comment line "php artisan serve" dilokasi folder recipes berada
6. Bukalah browser lalu masukan alamat "localhost:8000/recipes"
7. Untuk akses user admin masukan alamat "localhost:8000/dashboard"
